Combines **`ArrayList`**, **`HashMap`** (inside `ScanTally`), **`TreeMap`** (for sorted print in `CollectionsFinalApp`), and **`LinkedHashSet`** (inside `FirstSeenOrder`).

Open each `.java` file under `src/main/java/learning/collections/module3final/`; the tasks for **that file** are in the block comment at the top.

