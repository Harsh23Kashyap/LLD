# Design thinking — mini practice

**Goal:** Use **composition** (`Laptop` **HAS-A** `Cpu` and **HAS-A** `Disk`), not `extends` on those parts.

Open each `.java` under `src/main/java/learning/design/practice/`; tasks are in the top comment (`Cpu` / `Disk` are given; you fill **`Laptop`** and **`DesignPracticeApp`**).

Run **`./run.sh`** when done. Expected:

```text
cpu tick
disk 512 GB
laptop booted
```
